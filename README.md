# CodeAlpha C++ Internship Projects

This repository contains the C++ projects I completed during my internship at **CodeAlpha**. These projects helped me strengthen my understanding of C++ programming, object-oriented programming, file handling, arrays, functions, and basic data structures.

---

## Projects

### 1. Semester GPA & CGPA Calculator

### Description
A C++ console application that calculates the GPA for multiple semesters and the final CGPA. The user enters course names, grades, and credit hours for each semester, and the program calculates semester GPA as well as the overall CGPA.

### Features
- Calculate Semester GPA
- Calculate Overall CGPA
- Multiple semester support
- Course-wise grade entry
- Credit-hour based calculation

---

### 2. Login & Registration System

### Description
A simple authentication system developed in C++. The project allows users to register and log in using a username and password. User credentials are stored using text files.

### Features
- User Registration
- User Login
- Username availability check
- File handling for storing credentials
- Password verification

---

### 3. Banking Management System

### Description
A console-based banking system built using Object-Oriented Programming concepts. The application allows customers to perform banking operations such as deposits, withdrawals, fund transfers, and viewing transaction history.

### Features
- Customer Registration
- Deposit Money
- Withdraw Money
- Transfer Funds
- View Account Information
- Transaction History
- Object-Oriented Design using Classes
